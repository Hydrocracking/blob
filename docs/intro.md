# 欢迎来到 Hydrocracking 的知识库

这是一个个人知识库，包含技术笔记、投资记录和日常随笔。

## 内容结构

- **文档 (Docs)** — 结构化知识库，按分类整理
  - `/docs/tech/` — 技术笔记
  - `/docs/investment/` — 投资研究
- **博客 (Blog)** — 时间线笔记、随笔、读书笔记

## 使用说明

所有内容以 Markdown 格式存储，托管在 GitHub 私有仓库。

如需添加新内容，请遵循以下格式：

- 技术笔记 → `docs/tech/`
- 投资笔记 → `docs/investment/`
- 博客文章 → `blog/`（文件命名：`YYYY-MM-DD-title.md`）
